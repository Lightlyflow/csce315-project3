## Connect to Host
1. Make sure to copy the PEM file in our Google Drive into the `hosting` folder
2. Make sure you are in the hosting directory
3. If you are unix:
   1. `chmod u+x connect.sh`
   2. `./connect.sh`
4. If you are windows (WIP, may not work):
   1. `./connect.bat`
